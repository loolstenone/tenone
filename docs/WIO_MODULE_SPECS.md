# WIO 모듈 세부 기능 명세

> 27개 모듈 × 세부 기능. WIO가 만들어야 할 모든 것.

---

## 공통 모듈 (5개)

### 1. Home — 대시보드

| 기능 | 설명 |
|------|------|
| 오늘 할 일 | 마감 임박 업무, 미처리 알림 |
| 알림 센터 | 멘션, 결재 요청, 코멘트, 초대 |
| 빠른 액션 | 새 프로젝트, 새 글, 시수 입력 바로가기 |
| KPI 카드 | 테넌트 핵심 지표 4~6개 (커스텀 가능) |
| 최근 활동 | 내가 참여한 프로젝트/글/댓글 타임라인 |
| 팀원 현황 | 온라인 멤버, 오늘 생일, 입사 기념일 |
| 일정 위젯 | 오늘/이번 주 일정 미니 캘린더 |
| 날씨/격언 | 오늘의 한 마디 (Ten:One Principle) |

---

### 2. People — 멤버 관리

| 기능 | 설명 |
|------|------|
| 멤버 목록 | 검색, 필터 (역할/부서/스킬/등급) |
| 프로필 상세 | 이름, 소속, 역할, 스킬 태그, 자기소개 |
| 역할 관리 | owner/admin/manager/member/guest 변경 |
| 등급/포인트 | 활동 포인트, 등급 (Bronze→Diamond) |
| 활동 이력 | 참여 프로젝트, 작성 글, 교육 수료 |
| 가입/초대 | 이메일 초대, 초대 링크 생성, 가입 승인 |
| 그룹/태그 | 기수, 부서, 팀, 소속별 그룹핑 |
| 탈퇴/비활성 | 탈퇴 처리, 비활성 멤버 관리 |
| 프로필 카드 | 외부 공유용 프로필 카드 (Badak 스타 등) |
| 내보내기 | CSV/Excel 멤버 목록 다운로드 |

### 3. Talk — 소통

| 기능 | 설명 |
|------|------|
| 공지사항 | 고정 공지, 필독 표시, 확인 체크 |
| 자유게시판 | 글/댓글/대댓글, 좋아요, 첨부파일 |
| Q&A | 질문/답변, 채택, 태그 |
| 일정/이벤트 | 캘린더, 이벤트 생성, 참가 신청, 리마인더 |
| 채팅 | 1:1 + 그룹 채팅, 파일 공유 |
| 투표/설문 | 간편 투표, 설문 작성/결과 |
| 카테고리 | 게시판별 카테고리 (업종별, 기수별 등) |
| 알림 설정 | 게시판별 알림 on/off |
| 검색 | 전체 게시글/댓글 풀텍스트 검색 |
| 신고/관리 | 부적절 글 신고, 관리자 삭제/숨김 |

### 4. Content — 콘텐츠 관리

|------|------|
| 글 작성 | 리치텍스트 에디터 (Tiptap) |

| 버전 관리 | 수정 이력, 되돌리기 |
| 공동 작성 | 여러 작성자 협업 |
| 통계 | 조회수, 체류 시간, 공유 수 |
| 승인 워크플로우 | 초안→검수→승인→발행 |
