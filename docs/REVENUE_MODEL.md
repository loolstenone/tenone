# Ten:One™ Universe 수익 모델

> 각 서비스는 자기만의 수익 구조를 갖는다.
> WIO는 WIO의 가격이, SmarComm은 SmarComm의 가격이 있다.

---

## 수익 서비스 지도

```
Ten:One™ Universe
├── 💰 수익 서비스 (직접 매출)
│   ├── WIO          — SaaS 구독 + 구축 컨설팅
│   ├── SmarComm     — 마케팅 대행 + AI 솔루션
│   ├── HeRo         — 인재 매칭 수수료 + 컨설팅
│   ├── Brand Gravity — 브랜딩 컨설팅 프로젝트
│   ├── Naming Factory— 네이밍 프로젝트 + 네임드림
│   ├── Evolution School — 교육 수강료
│   └── Planner's    — 도구/굿즈 판매 + 워크숍
│
├── 🌱 성장 서비스 (간접 매출 / 인재 파이프라인)
│   ├── MADLeague    — 회비 + 스폰서 + 인재 공급
│   ├── Badak        — 유료 멤버십 + 이벤트 + 멘토 공급
│   ├── ChangeUp     — 교육 프로그램 수탁
│   └── domo         — B2B 연결 수수료
│
├── 📢 브랜딩 서비스 (트래픽 / 콘텐츠)
│   ├── Mindle       — 구독 + 스폰서 + 데이터 판매
│   ├── RooK         — AI 콘텐츠 제작 대행
│   ├── 0gamja       — 굿즈 + 광고
│   └── FWN          — 패션 콘텐츠 + 네트워크
│
└── 🏠 커뮤니티 서비스 (장기 자산)
    ├── Townity       — 지역 광고 + 공동구매
    ├── Seoul360      — 여행 콘텐츠 + 제휴
    ├── Mullaesian    — 지역 문화 프로젝트
    └── 기타          — 브랜드 자산 축적
```

## 가격 정책 원칙

1. **각 서비스는 독립 가격** — WIO 가격 ≠ SmarComm 가격 ≠ HeRo 가격
2. **Universe 번들 할인** — 2개 이상 서비스 동시 사용 시 10~30% 할인
3. **Free 티어 필수** — 모든 서비스에 무료 체험 제공 (전환율 확보)
4. **컨설팅은 별도** — SaaS 가격과 컨설팅 가격 분리
5. **사람·돈·시간 기반 과금** — 인원수, 거래량, 사용 시간에 비례
